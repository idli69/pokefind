# PokeFind - "Who's that Pokemon"
---

> A simple game i made with `Svelte 5` and `Tailwind CSS`.

## Tools used
---

- Svelte 5
- Tailwind CSS
- Vite
- Vercel 
- PokeAPI

## Installation

```
git clone https://github.com/idli69/pokefind.git
npm install 
npm run dev
```
